python模块

ConfigParser

该模块适用于配置文件的格式与windows ini文件类似，可以包含一个或多个节（section），每个节可以有多个参数（键=值）。可以更方便读取配置文件，将配置写入同一文件便于管理

os

该模块提供了非常丰富的方法用来处理文件和目录以及一些方便使用操作系统相关功能的函数

logging

主要用于进行格式化内容输出，可将格式化内容输出到文件，也可输出到屏幕，最重要的是可以让你跟踪代码运行时的事件,当程序崩溃时可以查看日志并且发现是什么引发了错误

re、time

re模块为Python的内置模块,Python程序中通过这个模块来使用正则表达式。

time模块也是python的内置模块,用于处理时间问题,提供了一系列的操作时间的函数。

subprocess

该模块允许你去创建一个新的进程让其执行另外的程序，并与它进行通信，获取标准的输入、标准输出、标准错误以及返回码等。

smtplib

该模块可以模拟一个smtp客户端实现邮件的发送功能。

json

该模块将字符串形式的json数据转化为字典，也可以将Python中的字典数据转化为字符串形式的json数据。

nmap

是一个帮助使用nmap端口扫描器的python库。它允许轻松操纵nmap扫描结果，也支持nmap脚本输出。
